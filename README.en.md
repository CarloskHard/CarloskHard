## 🚀 Featured Projects

### 🍽️[Platorama](#) *(In development - 2025)*
**The project:** My most ambitious personal project. A gastronomic social network that creates a space for users to manage their favorite spots and seek recommendations from friends.
**Technology:** Developed using *Clean Architecture* to share business logic and presentation across platforms.
*   **Frontend/App:** Android Studio + Compose Multiplatform + Kotlin.
*   **Backend/Auth:** Firebase (NoSQL and authentication).
*   **Local DB:** SQLite with Room as an abstraction layer.

### 🎨[Web Portfolio and E-Commerce (La Bicha Tattoo)](https://labichatattoo.es/) *(2025)*
**The project:** Freelance development of a complete platform for a tattoo artist, including an appointment system, an SEO-optimized portfolio, and an online store.
**Technology:** WordPress + WooCommerce.
*   *Note:* An example proving that in simple, fast-deployment projects (2 days), true power lies in knowing how to use established tools (CMS) to provide the best solution for the client without reinventing the wheel.
