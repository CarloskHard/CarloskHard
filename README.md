## 🚀 Proyectos Destacados

### 🍽️ [Platorama](#) *(En desarrollo - 2025)*
**El proyecto:** Mi proyecto personal más ambicioso. Una red social gastronómica que crea un espacio para que los usuarios gestionen sus locales favoritos y busquen recomendaciones de sus amistades.
**Tecnología:** Desarrollado bajo una *Clean Architecture* para reciclar lógica de negocio y presentación entre plataformas.
*   **Frontend/App:** Android Studio + Compose Multiplatform + Kotlin.
*   **Backend/Auth:** Firebase (NoSQL y autenticación).
*   **Local DB:** SQLite con Room como capa de abstracción.

### 🎨[Portfolio Web y E-Commerce (La Bicha Tattoo)](https://labichatattoo.es/) *(2025)*
**El proyecto:** Desarrollo freelance de una plataforma completa para una artista tatuadora, incluyendo sistema de citas, portafolio optimizado para SEO y tienda online.
**Tecnología:** WordPress + WooCommerce.
*   *Nota:* Un ejemplo de que en proyectos sencillos y de rápido despliegue (2 días), el verdadero poder reside en saber usar herramientas ya establecidas (CMS) para dar la mejor solución al cliente sin reinventar la rueda.
